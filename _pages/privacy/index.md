本プライバシーポリシーは Project Thelxie（以下、「本サイト」）がユーザーから収集するプライバシー情報及びその利用・保護方法について説明するものです。
本サイトにアクセスした時点で、ユーザーは本プライバシーポリシーに同意したものとみなされます。

## プライバシー情報の定義

プライバシー情報とは、個人情報、履歴情報及び特性情報を指します。
プライバシー情報のうち「個人情報」とは、個人情報の保護に関する法律にいう「個人情報」を指すものとし、生存する個人に関する情報であって、当該情報に含まれる氏名、生年月日その他の記述等により特定の個人を識別できる情報を指します。
プライバシー情報のうち「履歴情報及び特性情報」とは、上記に定める「個人情報」以外のものを指し、ご利用いただいたサービスやご覧になったページ、広告の履歴、ユーザーが検索された検索キーワード、ご利用日時、ご利用の方法、ご利用環境、ユーザーの IP アドレス、Cookie、端末の個体識別情報など、ユーザーが本サイトを利用する際の履歴に関する情報や、ユーザーの有する特性に関する情報を指します。

## 収集する情報について

本サイトでは、上記に定めるプライバシー情報を一切収集しておりません。
ただし、下記に定めるアクセス解析サービスによって、本サイトに関するトラフィックデータを収集する場合がございます。このデータは匿名で処理されており、個人を特定することはございません。

## アクセス解析サービスについて

本サイトでは、アクセス状況を解析するために Cloudflare, Inc. が提供する [Cloudflare Web Analytics](https://www.cloudflare.com/ja-jp/web-analytics/) を利用する場合がございます。 このサービスは、Cookie や localStorage 等のクライアントサイドの状態を一切使用せずに、本サイトに関するトラフィックデータを収集いたします。また、収集の過程で IP アドレス、User-Agent、その他のデータを介してフィンガープリントを取得することもございません。
Cloudflare Web Analytics に関する詳細は、[Cloudflare の利用規約](https://www.cloudflare.com/ja-jp/website-terms/)及び [Cloudflare のプライバシーポリシー](https://www.cloudflare.com/ja-jp/privacypolicy/)をご確認ください。

## プライバシーポリシーの変更について

本プライバシポリシーは予告なく変更される可能性がございます。
変更後のプライバシーポリシーは、本ページへの掲載時に効力を発します。
なお、本プライバシーポリシーの改訂履歴は [GitHub](https://github.com/yutasyndrome/project-thelxie-contents/commits/main/_pages/privacy/index.md) で確認することができます。
