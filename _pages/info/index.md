## Project Thelxie について

Project Thelxie（以下、「本サイト」）は非公式の Web サイトであり、いかなる組織とも提携しておりません。  
本サイト上に掲載されているゲーム関連コンテンツの著作権はすべて COGNOSPHERE PTE. LTD. に帰属いたします。
本サイト上に掲載されている内容を無断で転載することを禁止いたします。

## リンク及び引用について

当サイトはリンクフリーであり、リンクを掲載する際の許可や連絡は不要です。
ただし、画像ファイルへの直リンクやインラインフレームの使用はご遠慮ください。
引用を行う際は、引用元の明記及び該当ページへのリンク掲載をお願いいたします。

## 推奨環境

本サイトでは多数の画像を掲載しているため、ファイルサイズの軽量化及び表示速度の短縮化を目的として、画像フォーマットに WebP を採用しております。フォールバックは行っておりませんので、下記の WebP 対応ブラウザの使用を強く推奨いたします。
（参考：[よくある質問 | WebP | Google for Developers](https://developers.google.com/speed/webp/faq?hl=ja)）

- Google Chrome（パソコン）17 以降
- Google Chrome for Android バージョン 25 以降
- Microsoft Edge 18+
- Firefox 65 以降
- Opera 11.10 以降
- ネイティブ ウェブブラウザ、Android 4.0+（ICS）

ただし、Safari は下記の「データ保存について」における理由により、非推奨です。

## データ保存について

本サイトでは、データの保存にブラウザの localStorage（ローカルストレージ）を使用しております。
localStorage のデータはブラウザを閉じても保持され、ユーザー側による明示的な削除が行われない限り永続的に保存されます。
ただし、Safari に限り localStorage の保存期間が 7 日間に制限されております。
（参考：[Full Third-Party Cookie Blocking and More | WebKit](https://webkit.org/blog/10218/full-third-party-cookie-blocking-and-more/)）
そのため、Safari 以外のブラウザの使用を推奨いたします。
また、ブラウザのシークレット（プライベート）モードでは、localStorage のデータはブラウザを閉じると削除されるため、通常モードの使用を推奨いたします。
